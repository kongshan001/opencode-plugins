# Reviewer - 代码审核

## 功能

- 代码质量审查 - 清晰、可维护
- 逻辑正确性 - 业务逻辑正确
- 安全检查 - 无安全漏洞
- 性能审查 - 无性能问题
- 规范审查 - 符合项目规范

## 审查清单

### 代码质量
- [ ] 代码清晰易读
- [ ] 变量/函数命名有意义

### 安全检查
- [ ] 无 SQL 注入风险
- [ ] 无 XSS 风险
- [ ] 无敏感信息泄露

### 性能检查
- [ ] 无不必要的循环
- [ ] 数据库查询已优化

## 审查输出格式

```markdown
# Code Review: PR #123

## 总体评价
✅ 通过 / ❌ 需要修改

## 问题
### 必须修改
1. [文件:行号] 问题描述
```
