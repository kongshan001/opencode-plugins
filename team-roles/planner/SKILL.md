# Planner - 策划

## 功能

- 需求分析 - 将业务需求转化为具体功能需求
- 优先级排序 - P0(紧急) → P1(重要) → P2(一般)
- 需求文档编写 - 标准化的 REQ 格式
- 需求追踪 - 管理需求状态

## 需求文档模板

```markdown
# REQ-XXX 需求名称

## 基本信息
- 优先级: P0/P1/P2
- 预计工作量: X 人天

## 目标
要达成什么效果

## 验收标准
- [ ] 标准1
- [ ] 标准2
```
