# Coordinator - 团队协调者

## 功能

- 协调策划、程序、审核、QA、文档各角色
- 管理完整开发流程
- 质量把控
- 进度追踪

## 工作流程

```
策划 → 程序 → 审核 → QA → 文档 → 提交推送
```

## 质量标准

| 阶段 | 标准 |
|------|------|
| 需求 | 每个 REQ 有验收标准 |
| 开发 | 代码可运行、无明显问题 |
| Review | 至少 1 人 approve |
| 测试 | 覆盖率达标 |
| 文档 | 与代码同步 |
| 提交 | 推送到远程仓库 |
